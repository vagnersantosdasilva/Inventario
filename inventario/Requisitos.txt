- Problema : Para que uma máquina cliente seja registrada no servidor pela primeira vez
			 - Verificar se máquina já foi registrada com outro nome - Podemos fazer isso utilizando o endereço mac 
			 	- caso positivo : atualizar nome
			 	- caso negativo : registrar normalmente.	

			Caso a máquina não tenha registro será negado acesso até que o programa cliente seja instalado e seja crido 
			o registro. 
			

Primeira idéia:
	- Maquina cliente solicita instalação
	- servidor verifica se maquina já tem registro
	- caso ainda não , envia ao cliente numero de registro
	- cliente recebe registro e grava essa informação , em seguida responde ao servidor que tudo ok
	- servidor finaliza processo de inclusão de maquina e responde ao cliente que tudo ok
	- caso o cliente não receba informação solicita novamente validação.
